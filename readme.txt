Adds a sidebar widget to search for song lyrics on the LyricWiki.org site.

== Description ==

[LyricWiki](http://lyricwiki.org "Lyrics From LyricWiki") is a free site which is a source where
anyone can go to get reliable lyrics for any song from any artist without being hammered by invasive ads.

This plugin provides a widget to place a search box right on the sidebar of your WordPress site so
that your readers can utilize the LyricWiki site and find their favorite lyrics from your blog!

== Installation ==

After downloading the zip file, unzip and follow these few simple steps:

1. Upload `LyricWikiSearch.php` to the `/wp-content/plugins/` or `/wp-content/plugins/widgets/`
directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag and drop your LyricWiki Search widgets onto the active sidebars of your choice on the
Presentation -> Sidebar Widgets administration screen
4. (Optional)Configure your LyricWiki Search widget and save your changes.

== Configuration ==

1. **Title** - As with most widgets you can provide a *title* to display above the LyricWiki Search

2. **Button Text** - You can configure the text of the search button if you like.

3. **Hide Help** - Check to hide the small link to help
